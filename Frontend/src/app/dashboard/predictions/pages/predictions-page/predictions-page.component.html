Modelos de Aprendizaje Utilizados

En nuestro proyecto, hemos utilizado dos modelos de aprendizaje supervisado para realizar predicciones en el ámbito del fútbol: RandomForestRegressor y LinearRegression, ambos disponibles en la biblioteca Stickylearn.

RandomForestRegressor

RandomForestRegressor es un algoritmo de aprendizaje supervisado que utiliza la técnica de "bosques aleatorios" para realizar predicciones. Este modelo construye múltiples árboles de decisión durante el entrenamiento y combina sus predicciones para mejorar la precisión y evitar el sobreajuste.

En nuestro proyecto, hemos aplicado RandomForestRegressor para predecir diversas métricas relacionadas con el rendimiento de los jugadores de fútbol, como el número de goles esperadas o asistencias esperadas

LinearRegression

LinearRegression es un modelo clásico de aprendizaje supervisado que busca establecer una relación lineal entre las variables de entrada y la variable objetivo. Este modelo asume que existe una relación lineal entre las características y la variable objetivo, y utiliza esta relación para realizar predicciones.

En nuestro proyecto, hemos utilizado LinearRegression para analizar y predecir el rendimiento de los jugadores de fútbol, por ejemplo para predecir el número de entradas esperadas en un partido, regates esperados o ACG (Acciones de Contribución de Goles).
